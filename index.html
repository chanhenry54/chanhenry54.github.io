Projects

Hechan Bot

Hechan Bot is an interactive bot for the popular chat application Discord that serves 600+ users. It features several text interactions in response to certain messages typed by the user, as well as some commands that allow the user to view various information about their player profile on the popular video game League of Legends.

Technologies used: JavaScript, Node.js, Discord API, Riot Games API, MongoDB, Git
